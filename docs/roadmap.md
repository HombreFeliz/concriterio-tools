# Roadmap — concriterio.tools (landing principal)

## v1 (lo que Claude Code debe construir ahora)

- Página única con header, hero, filtros por tipo, grid de tarjetas 2 columnas, footer con banners
- 4 herramientas iniciales: Resend, Mapbox GL JS, Liveblocks, UploadThing
- Tarjetas con toda la información visible (sin accordion)
- Filtrado client-side con vanilla JS
- CSS patterns generativos como cover de cada herramienta (placeholder hasta que haya capturas)
- Responsive: 2 columnas desktop, 1 columna móvil
- Favicon SVG con la T violeta
- Meta tags para SEO básico y OG

## v2 (mejoras posibles si el directorio crece)

- Capturas de pantalla reales de las demos como imagen de portada
- Filtro por categoría además de por tipo
- RSS feed para que herramientas de terceros puedan consumir el directorio
- Página /changelog o sección "recién añadidas" si el volumen supera las 20 herramientas
